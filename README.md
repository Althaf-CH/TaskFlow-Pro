## 👨‍💻 About

TaskFlow Pro is a modern full-stack task management system developed by **Mohammed Althaf C H** as part of a Full Stack Development Internship.

The application allows users to register, log in securely, create tasks, update task details, track progress, manage priorities, and monitor task completion through a professional dashboard interface.

---

## ✨ Features

### Authentication System

* User Registration
* User Login
* Session management using Local Storage
* Logout functionality

### Task Management

* Create new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Priority management (Low, Medium, High)

### Dashboard Features

* Total Tasks Counter
* Pending Tasks Counter
* Completed Tasks Counter
* High Priority Tasks Counter
* Task Completion Progress Bar
* Search Tasks
* Filter Tasks (All / Pending / Completed)
* Due Soon Highlighting
* Empty State Message
* Confirmation Before Delete

### User Experience

* Responsive Dashboard Design
* Modern Gradient UI
* Professional Task Cards
* Interactive Buttons
* Clean Navigation Layout

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm
* Modern Web Browser

### Installation

Clone the repository:

```bash
git clone https://github.com/Althaf-CH/taskflow-pro.git
```

Navigate into the project folder:

```bash
cd taskflow-pro
```

Install dependencies:

```bash
npm install
```

Start the backend server:

```bash
node server.js
```

Open the frontend files using Live Server or your browser.
